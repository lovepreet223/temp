Scenario Modeling Snippet code


def eps(revenue, margin, shares):
    """Simple EPS model."""
    operating_income = revenue * margin
    tax_rate = 0.21
    net_income = operating_income * (1 - tax_rate)
    return net_income / shares

# Example:
print(eps(1_250_000_000, 0.22, 187_500_000))


// Growth projection
const proj = (rev, growth, q=4) =>
  Array.from({length: q}, (_,i) => rev * (1 + growth)**(i+1));

Financial Formulas

Present Value of a cash flow stream:

\\[ PV = \sum_{t=1}^{T} \frac{CF_t}{(1 + r)^t} \\]

Dividend Discount Model (Gordon Growth):

\\[ P_0 = \frac{D_1}{r - g}, \quad r > g \\]

WACC:

\\[ \mathrm{WACC} = \frac{E}{D+E}R_e + \frac{D}{D+E}R_d(1 - \tau) \\]
